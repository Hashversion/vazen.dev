## Workspace + Build

- `pnpm` for package management.
- `turborepo` for task orchestration and filtering.

## Code Quality

- `eslint` for linting.
- `prettier` for formatting.
- `typescript` for strict typing.
- `knip` for dead-code/dependency detection.

## Git Workflow

- `lefthook` for fast hooks.
- `commitlint` with Conventional Commits in `commit-msg`.

## Local Infrastructure

- Docker Compose for local PostgreSQL.

## Runtime Safety

- `@t3-oss/env-nextjs` for typed env validation.
- `@nosecone/next` for security headers and CSP policies.
