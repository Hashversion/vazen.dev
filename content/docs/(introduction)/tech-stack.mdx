## Frontend

- Next.js (App Router)
- React 19
- TypeScript
- Tailwind CSS v4
- Motion

## API + Data

- oRPC (`@orpc/server`, `@orpc/client`, OpenAPI plugins)
- TanStack Query
- PostgreSQL
- Drizzle ORM + Drizzle Kit

## Auth + Security

- Better Auth
- Nosecone security headers + CSP middleware
- c15t consent manager

## Payments + AI + Messaging

- Polar (Better Auth integration + webhooks/checkout)
- Workers AI provider wrapper
- React Email templates

## Analytics + Observability

- PostHog
- Sentry (`@sentry/nextjs`)

## Platform + Tooling

- Cloudflare Workers via OpenNext
- Wrangler for bindings and environments
- Turborepo + pnpm workspaces
